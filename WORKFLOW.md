## 📊 Workflows  

To add extra features, follow these steps:  

✅ Update `config.yaml`  
🔐 Update `secrets.yaml` [Optional]  
⚙️ Update `params.yaml`  
🗄 Update the entity  
🔧 Update the configuration manager in `src/config`  
➕ Update the components  
➖ Update the pipeline  
🔄 Update `main.py`  
🕒 Update `dvc.yaml`  
🚀 Update `app.py`
